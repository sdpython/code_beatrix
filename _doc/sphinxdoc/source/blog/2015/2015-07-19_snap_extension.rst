    `Snap <http://snap.berkeley.edu/>`_ est une version
    javascript de
    `Scratch <https://scratch.mit.edu/>`_.
    Le site suivant recense quelques extensions ou déclinaisons :
    `Awesome Snap! <https://github.com/Gubolin/awesome-snap/blob/master/README.md>`_.
    Parmi elles, on trouve `BeetleBlocks <http://beetleblocks.com/>`_ qui fonctionne
    en trois dimensions eu lieu de deux.
    Il a presque la même interface
    `interface <http://beetleblocks.com/run/>`_.
    Une autre déclinaison propose d'utiliser
    Snap pour faire de la broderie
    `turtlestitch <http://turtlestitch.org/>`_.
